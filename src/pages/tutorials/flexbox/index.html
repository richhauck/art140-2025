A CSS flex box is an essential building block of HTML layouts, as it can create rows or columns and transition between the two.

First, add a div with a class name of "flex" to your HTML:

{% highlight "html" %}
{% endhighlight %}

In your CSS, we'll add a class called 'flex':

{% highlight "css" %} .flex{ display: flex; } {% endhighlight %}

Using a class to handle a specific property like this is often called a "utility" class. Instead of baking the behavior into a specific layout element, we can tag anything we want to behave with this simple selector.

Now, any child elements you add to this container will behave as columns. For instance, to create a two-column layout:

{% highlight "html" %}
1
2
{% endhighlight %}
