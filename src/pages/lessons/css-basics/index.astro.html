Cascading Style Sheets (CSS) is a language used to define the presentation and style of HTML.

Where CSS Resides

CSS can reside within three places:

  1. Inline via a style attribute: {% highlight "html" %}

    This is a styled header

    {% endhighlight %}
  2. Within the page within a head tag: {% highlight "html" %} ... ... {% endhighlight %}
  3. Within the page within a head tag: {% highlight "html" %} ... ... ... {% endhighlight %}
In CSS, "the closest rule wins", so the style definition closest to the markup is what rules.

Reset

Not all browsers render the same way which has made it a headache for web designers. The open source community has responded, and one way around this is by using CSS that helps override the default styling that comes with browsers.

  • Reset CSS - a tool created by Eric Meyer in 2007 to assist in making browsers behave the same.
  • Normalize.css - in some ways a successor to Reset. Makes browser behaviors consistent.
  • New CSS Reset

System Fonts

System fonts are fonts that are installed by default on the computer. Defining your website to use these fonts saves download time, as the user's computer doesn't need to download what's already installed on their machine.

  • CSS Web Safe Font Combinations (W3Schools)
