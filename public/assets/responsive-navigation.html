This basic example shows one approach to how a responsive nav can work using CSS an minimal JavaScript. We create a button with an id of 'toggle-nav-btn' that's placed absolutely. When clicked, it dynamically adds a class of 'nav-open' to the body tag. When this is in-place, our CSS kicks in to assign our #primary-nav with 'grid-template-rows: 1fr;' (which will scale the height to fit the contents). We also change the character in the button from a hamburger to a close.
